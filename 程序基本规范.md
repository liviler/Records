## 为什么要规范命名？
- 提高代码的可读性
- 提高可维护性
## 命名方法
### 名称的基本规则
- 名称一般由**字母**、**数字**、**下划**线组成
    Java 还可以包含**$**
- 不能以**数字**开头
- 变量名称不能是语言中的**保留字**(高级语言中已经定义过的字)
- 一般区分大小写
    PHP 的变量名区分大小写，但是函数名称不区分大小写
Fortran
- 变量名长度: 31个字符
- 第一个字符必须是字母
